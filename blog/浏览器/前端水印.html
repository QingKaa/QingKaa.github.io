给网页添加水印

实现原理

原理十分简单： 使用Canvas绘制水印文字，并将其转换成图片，生成的图片作为背景(or：直接插入这个图片作为背景，or 直接将Canvas插入，无需转换成图片)，以绝对定位方式添加到目标元素上。

此处是通过将canvas转换成图片(只生成小尺寸的canvas)，在将图片设置成背景（方便通过css的background-repeat属性来重复）

代码

大致流程：

  1. 创建canvas
  2. 绘制水印文字
  3. 通过canvas.toDataURL()将canvas转为图片
  4. 创建一个作为背景的div并设置样式（图片作为背景）
  5. 将div插入到目标元素去

以下是核心代码，做的完善一些的可以将一些变量（例如width，height等）通过参数方式传入。此处只作参考。

function createWatermark(targetEl: HTMLElement, content: string) {
    const canvas = document.createElement('canvas')
    const width = '150px'
    const height = '130px'
    canvas.setAttribute('width', width + 'px')
    canvas.setAttribute('height', height + 'px')

    const ctx = canvas.getContext('2d')!
    ctx.textAlign = 'center'
    ctx.textBaseline = 'middle'
    ctx.font = '20px Arial'
    ctx.fillStyle = 'rgba(0,0,0,0.5)'
    ctx.rotate(Math.PI / 180 * 30);
    ctx.fillText(content, 50, 50);

    const targetZIndex = targetEl.style.zIndex || 1000
    const base64Url = canvas.toDataURL();
    const renderId = `__waterMark_instance_${Date.now()}`
    const _waterMark = document.querySelector(`.${renderId}`)

    const waterMark = _waterMark || document.createElement('div')
    const bgDivStyleStr = `
        position: absolute;
        top:0;
        left:0;
        width:100%;
        height:100%;
        pointer-events:none;
        background-repeat: repeat;
        z-index:${targetZIndex};
        background-image: url(${base64Url});
    `
    waterMark.setAttribute('style', bgDivStyleStr);
    waterMark.classList.add(renderId);

    if (!_waterMark) {
        targetEl.style.position = 'relative';
        targetEl.insertBefore(waterMark, targetEl.firstChild);
    }
}

避免通过调试隐藏水印

是的，浏览器打开调试十分简单就可隐藏一个元素，例如：

  1. F12打开调试，选中水印元素，右键，选择删除，即可隐藏水印。
  2. F12打开调试，选中水印元素，轻按一下H，水印就不见了。
  3. ...

所谓水印不过一句玩笑~~

避免这种情况，需要监听元素的显示、隐藏、删除等状态变化，并作出相应的处理。 是的，没错，用的是 MutationObserver 这个浏览器的api。

MutationObserver 接口提供了监视对 DOM 树所做更改的能力。它被设计为旧的 Mutation Events 功能的替代品，该功能是 DOM3 Events 规范的一部分。MDN-MutationObserveropen in new window

增加MutationObserver（完整代码）

function createWatermark(targetEl: HTMLElement, content: string) {
    const canvas = document.createElement('canvas')
    const width = '150px'
    const height = '130px'
    canvas.setAttribute('width', width + 'px')
    canvas.setAttribute('height', height + 'px')

    const ctx = canvas.getContext('2d')!
    ctx.textAlign = 'center'
    ctx.textBaseline = 'middle'
    ctx.font = '20px Arial'
    ctx.fillStyle = 'rgba(0,0,0,0.5)'
    ctx.rotate(Math.PI / 180 * 30);
    ctx.fillText(content, 50, 50);

    const targetZIndex = targetEl.style.zIndex || 1000
    const base64Url = canvas.toDataURL();
    const renderId = `__waterMark_instance_${Date.now()}`
    const _waterMark = document.querySelector(`.${renderId}`)

    const waterMark = _waterMark || document.createElement('div')
    const bgDivStyleStr = `
        position: absolute;
        top:0;
        left:0;
        width:100%;
        height:100%;
        pointer-events:none;
        background-repeat: repeat;
        z-index:${targetZIndex};
        background-image: url(${base64Url});
    `
    waterMark.setAttribute('style', bgDivStyleStr);
    waterMark.classList.add(renderId);


    if (!_waterMark) {
        targetEl.style.position = 'relative';
        targetEl.insertBefore(waterMark, targetEl.firstChild);
    }

    // @ts-ignore
    const MutationObserver = window.MutationObserver || window.WebKitMutationObserver;
    if (MutationObserver) {
        let mo = new MutationObserver(function () {
            const __waterMark = document.querySelector(`.${renderId}`);
            let needToReCreate = !__waterMark
            // 当元素存在时候，判断display、visibility的值以及style 属性判断是否需要重新创建
            if (__waterMark) {
                const style = window.getComputedStyle(__waterMark)
                const isUnSee = style.display == 'none' || style.visibility == 'hidden' || Number(style.opacity) < 0.1
                needToReCreate = isUnSee || __waterMark.getAttribute('style') !== bgDivStyleStr
                // 如果需要重新创建则先把旧的移除
                needToReCreate && moveWaterMark(targetEl, __waterMark as HTMLElement)
            }
            // 创建一个新的
            if (needToReCreate) {
                // 避免一直触发
                mo.disconnect();
                createWatermark(targetEl, content);
            }
        })

        mo.observe(targetEl, {
            attributes: true,
            subtree: true,
            childList: true
        })
    }
}

function moveWaterMark(parent: HTMLElement, target: HTMLElement) {
    try {
        parent.removeChild(target)
    } catch (error) {
        console.log(' moveWaterMark =====> error:', error);
    }
}

给图片添加水印

给图片添加水印也大同小异，在Canvas上将图片绘制出来，在绘制水印文字，然后把canvas转换成图片即可。

/**
 * 给图片右下角添加水印
 * @param imgSrc 
 * @param content 
 * @param cb 
 */
export function imageAddWatermark(imgSrc: string, options: ImgWatermarkOption, cb?: Function,) {
    const img = new Image();
    img.src = imgSrc;
    img.crossOrigin = 'anonymous';
    img.onload = function () {
        const canvas = document.createElement('canvas');
        canvas.width = img.width;
        canvas.height = img.height;
        const ctx = canvas.getContext('2d')!;

        ctx.drawImage(img, 0, 0);
        ctx.textAlign = 'right'
        ctx.textBaseline = 'bottom'
        ctx.font = '20px Arial'
        ctx.fillStyle = options.textColor || 'rgba(0,0,0,0.5)'
        ctx.fillText(options.content || '清咖', img.width - (options.offsetRight || 10), img.height - (options.offsetBottom || 10));

        const base64Url = canvas.toDataURL();
        cb && cb(base64Url);
    }
}
